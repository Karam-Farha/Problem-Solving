# Problem Description
Given 4 arrays of Integers how many tupels (i , j, k, l) there is such that:

    A[i] + B[j] + C[k] + D[l] == 0

## Constraint Questions
- All arrays have the same size

## Examples
### input: 	5 3 2 1 4
### ans:	5 & 4
______________

### input: 1 8 6 2 5 4 8 3 7
### ans: 8 & 7
